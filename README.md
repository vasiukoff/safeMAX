# safeMAX
# Изолированная среда для корпоративного мессенджера Max (Android)

Набор утилит и конфигурационных файлов для развертывания мессенджера Max в изолированном рабочем профиле (Work Profile) ОС Android с фильтрацией сетевого трафика.

## Содержимое репозитория

* `shelter.apk` — open-source интерфейс для управления рабочим профилем Android.
* `cromite.apk` — веб-браузер для загрузки обновлений внутри изолированного контура.
* `netguard.apk` — локальный firewall для контроля трафика (не требует root-прав).
* `max.apk` — установочный пакет клиента мессенджера.
* `netguard_xxx.xml` — профиль настроек firewall с правилами блокировки.
* `max_audit.cmd` — скрипт мониторинга системных логов (logcat) через ADB.

## Инструкция по развертыванию

### 1. Инициализация рабочего профиля
1. Установите `shelter.apk` в основной профиль устройства.
2. Запустите Shelter и следуйте системным инструкциям для инициализации рабочего окружения. В системном лаунчере появится вкладка «Рабочие» (Work).

### 2. Установка компонентов в изолированную среду
1. Откройте приложение Shelter.
2. В контекстном меню (три точки) выберите пункт **«Установить APK в Shelter»**.
3. Поочередно установите файлы `cromite.apk`, `netguard.apk` и `max.apk`. 
**Внимание:** Не запускайте установленные приложения до завершения настройки сетевого экрана.

### 3. Приостановка фоновых процессов
Внутри вкладки рабочего профиля в приложении Shelter выберите все сторонние и неиспользуемые системные приложения (кроме Cromite, NetGuard, Max и системного файлового менеджера) и примените к ним действие **«Заморозить» (Freeze)**.

### 4. Настройка правил маршрутизации
1. Скопируйте файл `netguard_max_rules.xml` во внутреннюю память рабочего профиля (или загрузите его повторно через установленный в профиле браузер Cromite).
2. Запустите NetGuard (из рабочего профиля).
3. Перейдите в **Меню ➔ Настройки ➔ Резервное копирование ➔ Импорт настроек**.
4. Укажите путь к файлу `netguard_xxx.xml`.
5. Активируйте фильтрацию трафика с помощью переключателя в левом верхнем углу главного экрана NetGuard. 
6. Убедитесь, что в правилах приложения Max заблокированы прямые соединения (индикаторы Wi-Fi и мобильной сети выделены красным).

### 5. Запуск
Запустите мессенджер Max из вкладки рабочего профиля и пройдите авторизацию. 
Из системных разрешений предоставьте приложению исключительно доступ к файловой системе для обмена документами. Контактная база формируется вручную через встроенный поиск.
